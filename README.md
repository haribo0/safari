## 프로젝트 소개
- safari는 중고거래 마켓인 '당근마켓을' 모티브로 기획되어 유저 간의 '거래'와 그 안에서 일어나는 '상호작용'에서 아이디어를 확장시켜 대여, 경매, 커뮤니티 기능을 담고 있는 종합 포털사이트.
